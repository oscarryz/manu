How to

  1. Create a Fleeting Notes Inbox

    • Fleeting notes are raw, unprocessed notes.

    • Keep references to source material if deemed necessary

    • They will be deleted once they're processed.

  2. Process Fleeting Notes and Create New Notes in the Permanent Notes Folder

    • Each new note should have a unique ID.

    • Notes can be atomic or hub:

      • Atomic: Keeps a single idea. Use your own words

      • Hub: Aggregates other notes and draws conclusions. This is where new knowledge is created.

  3. Create Tags and Backlinks as Needed

    • At any point, create tags and backlinks to connect notes.
