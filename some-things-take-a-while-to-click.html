Some things take a while to click

Often times, the first time I try to learn a new concept it is really hard. Nothing makes sense, and my mind tries to map it to something I already know.

Many times, that doesn’t quite work and I just think, I learn it later.

Depending on when I need it, I might put more effort and the practice would clarify things. Some other times, I just roll with it and don’t fully understand everything.

Some other times though, then I don’t need the knowledge, I just let it go, and months and even years later everything is very clear.


One “extreme” example is Haskell syntax - I say extreme because if programming languages are already specialized knowledge, when I first tried to learn it several years (if not decades by now) nothing made sense -

For instance: 

sum :: Num [a] => a

Yeah, right! (by the way, that says that `sum` is a function that takes an array - or list - of any type as long as they are `Num` type class and returns a value `a`, so sum takes an array of numbers and returns a number).

Anyway, the example is not as important, today I see that and I understand it clearly and even find it appealing because how succinct it is.
That’s just one example, the point is, some things just take a little longer to click in my brain, because there are a number of connections that need to be formed before, more context, other knowledge, things that look similar. Is surprising how years later I can try to re-learn something and everything starts making sense.
