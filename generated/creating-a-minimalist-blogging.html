Creating a minimalist blogging platform

That was the goal of this project; create a simple way to create blogs by simply start writing directly in the blog instead of having to go through an complex application or having to write the blog in a regular text editor and have it transformed to html with a static site generator.

The idea while good, turned out to be (as many things in programming) easier to say that actually building it. It wasn’t particularly hard, just a little bit cumbersome, mostly due the lack of time.

So, taking as “inspiration” the https://manuelmoreale.com/ blog, we came up with this “design”:

Which is basically the same design but adding editor menus to for new entries, edit them and publish them.

I’ll keep documenting the approach we took to write this down and how far we got. This right here is the first entry on the working platform, and while still far from complete is now usable.
