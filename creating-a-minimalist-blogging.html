Creating a minimalist blogging platform

That was the goal of this project; create a simple way to create blogs where you can just start writing directly into the blog instead of having to go through a complex application or have to write in a weird format and then transform the blog, like with static site generators.

The idea while good, turned out to be (as many things in programming) easier to say that actually building it. It wasn’t particularly hard, just a little bit cumbersome, mostly due the lack of time.

So, taking as “inspiration” the https://manuelmoreale.com/ blog, we came up with this “design”:

Which is basically the same design but adding editor menus to for new entries, edit them and publish them.
